Identyfikatory - ID

Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID to skrót od "identyfikator" - unikalny znacznik przypisany do elementu HTML
Jaki jest cel stosowania ID? ID służy do identyfikacji pojedynczego elementu na stronie w celu nadania mu unikalnych stylów lub manipulacji za pomocą JavaScript
Jakie jest znaczenie znaku # Znak # w CSS poprzedza nazwę identyfikatora, wskazując, że styl dotyczy elementu o danym ID
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS: #nazwa_id { style }
W HTML: <element id="nazwa_id">
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID musi być unikalne w obrębie całego dokumentu HTML
  2. Element może mieć tylko jedno ID
  3. ID jest używane do stylowania pojedynczych elementów lub manipulacji za pomocą JavaScript
